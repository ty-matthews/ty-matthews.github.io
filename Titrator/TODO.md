# pH indicators — exhaustive list (CRC reference data)

## NIST source — access status

**I was not able to generate a list using NIST data only.** The NIST publication page **“Indicators for Acids and Bases”** (Thomas J. Bruno and Paris D. N. Svoronos) returns **403 Forbidden** when accessed programmatically, so the NIST table could not be read or extracted.

That NIST-authored content appears as a chapter in the **CRC Handbook of Chemistry and Physics**, 93rd edition (2012): same type of exhaustive indicator table (pH ranges, color changes, preparation). To use **NIST data only**, you would need to consult that chapter directly (e.g. print or institutional access to CRC Handbook 93rd ed.) or the NIST publication if your network allows:  
[https://www.nist.gov/publications/indicators-acids-and-bases](https://www.nist.gov/publications/indicators-acids-and-bases)

## Source used for this list: CRC reference data

The list below is taken from **CRC reference data (acid–base indicators)** by **A. K. Covington**, as published in the PDF “acid-base indicators” (Bishop, E., Ed., *Indicators*, Pergamon, Oxford, 1972). That PDF is widely used in teaching and is part of the same CRC/Handbook family; it is **not** the NIST-authored table.

- **Reference:** CRC reference data — acid-base indicators (A. K. Covington).  
- **Hosted copy (course use):** [Colorado State — CRC reference data / acid-base indicators PDF](https://diverdi.colostate.edu/all_courses/CRC%20reference%20data/acid-base%20indicators.pdf)

Color codes in the original (expanded below): C = colorless, Y = yellow, R = red, B = blue, P = purple, O = orange, V = violet, Pk = pink, A = amber, B/G = blue-green.

Indicators with **two transition ranges** are listed as two rows (e.g. Cresol Red, Thymol Blue, Metacresol Purple, 2-(p-Dimethylaminophenylazo)pyridine, Alizarin).
